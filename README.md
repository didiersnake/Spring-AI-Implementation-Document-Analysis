# Document Analyzer with Gemini AI API

Spring Boot application with a REST API that reads sentiment analysis results from a file, use Gemini API to categorize feedback and provide actionable insights, and display a dashboard

## Objectives:
* Build a Spring Boot application that exposes a REST API.
* Read and process sentiment analysis results from file.
* Integrate the Gemini API for categorizing customer feedback.
* Display insights in a user-friendly dashboard for easy interpretation.
